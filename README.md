# AI_ReportingAutomation
Automating weekly test reporting to management via AI. This report generated following : 
1. SPrint Planned items- execution status (Table and Graph, SUmmarize).
2. Sprint Defect count by Severity in graph format.
3. Overall Project Regression script and Automation count. 
Data is collected from : Jiira board, Excel placed on sharepoint (Automation coverage/test cases count/completion count)
Limitation of Project : Did not use zephyr, x-ray or any plugin to create and execute test on jiira - they are recorded in Excel.
